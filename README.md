# Falex OS

Falex OS は、カスタマイズ性の高いLinuxデスクトップディストリビューションです。GNOMEデスクトップ環境をベースに、ガラスをイメージしたデザインを提供します。

## 特徴

- **デスクトップ環境**: GTK 3 と GNOME
- **カスタマイズ性**: タスクバー、スタートメニュー、タイトルバー、デスクトップウィジェット、通知センター、コントロールセンターなどの色（グラデーション、透明度を含む）と構成を細かくカスタマイズ可能
- **移行テンプレート**: Windows クラシック、Windows リニューアル、Ubuntu、Mac、Chrome OS、Linux Mint の各配置テンプレート設定
- **バージョン**: 初期バージョンコードネーム "Falex Aero"

## プロジェクト構造

- `build/`: ビルドスクリプト
- `config/`: 設定ファイル
- `themes/`: テーマファイル
- `templates/`: レイアウトテンプレート
- `docs/`: ドキュメント
- `assets/`: アセット（ロゴなど）
